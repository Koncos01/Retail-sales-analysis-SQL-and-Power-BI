# Regional Sales Analysis Project

## Overview
This project demonstrates a complete end-to-end data analytics workflow. It involves integrating disparate sales data from 5 different geographical regions, performing comprehensive data cleaning and transformation (ETL) using MySQL, and creating an interactive business dashboard in Power BI.

## Tech Stack
Database: MySQL (Data integration, cleaning, and modeling)

Visualization: Power BI Desktop

Language: SQL, DAX

## Key Features & Solutions
ETL Pipeline: Consolidated 5 regional CSV datasets into a unified database using UNION ALL.

Data Cleaning: Handled data type conversions (string to date/decimal), standardized category names, and ensured referential integrity.

Advanced Modeling: Created optimized SQL Views to serve as a clean data source for reporting.


Interactive Dashboard: Features include regional heatmaps, sales trend analysis, and KPI cards for executive-level insights..

## Project Structure
SQL_Transformation_and_Reporting.sql: Contains the full data pipeline from raw data to final analytics views.

Retail_Analytics_Dashboard.pbix: The Power BI file containing the data model and interactive visualizations.
