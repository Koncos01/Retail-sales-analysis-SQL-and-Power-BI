# Retail-sales-analysis-SQL-and-Power-BI
Regional sales analysis using MySQL for ETL and Power BI for visualization.

# Regional Sales Analysis Project

## Overview
Ez a projekt egy fiktív kiskereskedelmi hálózat 5 régiójának adatait dolgozza fel. 
A cél az volt, hogy az elszórt CSV adatokból egy egységes, tisztított SQL adatbázist és egy interaktív Power BI dashboardot hozzak létre.

## Felhasznált technológiák
- **SQL (MySQL):** Adattisztítás, UNION ALL, JOIN-ok, dátumformázás.
- **Power BI:** Adatmodellezés, vizualizáció

## Főbb funkciók
- 5 régió adatainak egyesítése egyetlen forrásba.
- Dinamikus üzleti rangsor a dashboardokon.
- Regionális értékesítési hőtérkép.
